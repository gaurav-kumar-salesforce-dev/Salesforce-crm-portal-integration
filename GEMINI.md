# CRM Portal

Project Type:
Salesforce-like CRM Portal

Tech Stack:
- Node.js
- Express
- PostgreSQL
- HTML
- CSS
- JavaScript

Requirements:
- Follow Salesforce Lightning UI patterns
- Maintain backward compatibility
- Use async/await
- Do not remove existing functionality
- Explain major architectural changes before implementation
- Keep UI consistent across all objects

Objects:
- Account
- Contact
- Lead
- Opportunity
- Case
- Campaign

Features:
- Chatter
- Activity Timeline
- Tasks
- Events
- Email Logging
- Related Lists
- Dynamic Forms
- Profile Based Permissions
